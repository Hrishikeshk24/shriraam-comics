🚀 Publish to Website

When you're happy with your changes, export the updated data.js file, replace the one in your folder, then push to GitHub to publish.

Publishing steps:
  1. Click Export data.js above and copy the code
  2. Open the data.js file in your folder and replace everything with the copied code
  3. Run in Terminal: cd ~/shriraam-comics && git add -A && git commit -m "new comic" && git push
  4. Your site updates at Shriraam.github.io in ~60 seconds ✅
